Hosting Database ASIP Online

Panduan langkah demi langkah mempublikasikan server & database ASIPCare secara gratis.

Pendahuluan

Aplikasi ASIPCare saat ini mendukung mode penyimpanan lokal (offline) dan mode online. Dengan menaruh database Anda di server online (cloud), Anda dapat:

  • Mengakses data pumping dari berbagai ponsel yang berbeda secara bersamaan.
  • Melakukan pencatatan kolaboratif bersama suami, pengasuh bayi, atau anggota keluarga lainnya.
  • Menjamin data tetap aman bahkan ketika Anda menghapus aplikasi atau mengganti ponsel.
PENTING: Kami telah memperbarui kode aplikasi Anda sehingga Anda dapat dengan mudah mengubah alamat URL server database langsung melalui halaman Pengaturan (Settings) aplikasi tanpa perlu mengompilasi ulang aplikasi!

Metode 1: Deploy Gratis Menggunakan Railway

Gratis Direkomendasikan (Mudah & Cepat)

Railway mendukung Node.js dan menyediakan fitur Persistent Volume. Ini sangat cocok untuk menyimpan database SQLite (database.sqlite) agar data tidak hilang ketika server dinyalakan ulang.

Langkah-langkah:

  1. Buka website Railway.app dan daftarlah dengan akun GitHub Anda.
  2. Hubungkan repositori GitHub Anda https://github.com/luviadibaskoro-beep/ASIP-MAS-LUVI.git.
  3. Pilih sub-folder backend/ sebagai root directory deployment.
  4. Pada tab Settings di project Railway Anda, tambahkan Volume (persistent disk) dan mount ke folder /data.
  5. Tambahkan Variabel Lingkungan (Environment Variables) berikut pada tab Variables:
    PORT=5000 DATABASE_PATH=/data/database.sqlite
  6. Deploy project Anda. Setelah berhasil, Railway akan menyediakan alamat URL publik berformat HTTPS, misalnya: https://asip-backend.up.railway.app.

Metode 2: Deploy Gratis Menggunakan Render

Gratis

Render menyediakan hosting gratis untuk server Node.js. Namun, untuk menjaga database SQLite Anda tetap aman di Render gratis, Anda direkomendasikan mengintegrasikannya dengan cloud database luar seperti Neon (Postgres) atau Supabase.

Langkah-langkah:

  1. Masuk ke Render.com dan buatlah akun menggunakan GitHub.
  2. Klik New + lalu pilih Web Service.
  3. Hubungkan repositori GitHub Anda dan atur kolom konfigurasi berikut:
    • Root Directory: backend
    • Build Command: npm install
    • Start Command: node server.js
  4. Gunakan database PostgreSQL gratis dari Neon.tech atau Supabase.
  5. Dapatkan koneksi string database Anda dan pasang pada variabel lingkungan DATABASE_URL di Render.
  6. Tunggu proses deployment selesai sampai Anda memperoleh link HTTPS publik dari Render, contoh: https://asip-backend.onrender.com.

Cara Menghubungkan Aplikasi ke Database Online Anda

Setelah mendapatkan URL publik dari salah satu layanan di atas, Anda tidak perlu mengompilasi ulang aplikasi! Cukup ikuti langkah berikut:

  1. Buka aplikasi ASIPCare di ponsel Android atau browser Anda.
  2. Pergi ke menu Pengaturan (Settings) (ikon gerigi).
  3. Cari kartu bertuliskan "Server Database Online" di sebelah kanan bawah.
  4. Masukkan alamat URL publik Anda (misalnya: https://asip-backend.up.railway.app) ke dalam kolom input.
  5. Klik tombol "Simpan & Hubungkan".
  6. Aplikasi akan secara otomatis mengetes koneksi ke server database online Anda. Jika berhasil, aplikasi akan melakukan memuat ulang (*reload*) dan mulai melakukan sinkronisasi data secara online!

Untuk kembali menggunakan database lokal offline, cukup kosongkan kolom input URL tersebut dan klik "Simpan & Hubungkan" kembali.
